3- Los TCs que fallan son el Tercero, Cuarto, Octavo y Noveno en total son 4 TCs.

4- En el archivo llamado create_kit_name_kit_test.py es donde estan los tcs automatizados
   para su ejecucion.

5- En el archivo data.py estan todos los diccionarios necesarios para mandar los request correctamente.

6- En el archivo configuration.py estan todas las URL necesarias para mandar los request correctamente
   segun la informacion en API Urban Grocers. (Nota: Tendras que actualizar la condfiguracion
   del URL_SERVICE con una vigente).

7- En el archivo sender_stand_request.py estan todas las funciones de los request que se llaman en
   el archivo create_kit_name_kit_test.py
